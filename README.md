# Smart Campus Navigation

A web-based campus navigation system using Google Maps and Firebase.

## Features
- Campus building markers
- Route visualization between buildings
- Distance & time estimation
- Firebase Realtime Database integration

## Tech Stack
- HTML, CSS, JavaScript
- Google Maps JavaScript API
- Firebase Realtime Database

## How to Run
1. Open `index.html` in a browser
2. Ensure a valid Google Maps API key is added
3. Firebase must be configured correctly
